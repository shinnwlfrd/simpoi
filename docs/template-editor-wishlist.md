# Wishlist Editor Template Tiptap

## Sudah Dikerjakan

- Dropdown placeholder siap pakai untuk semua field surat.
- Blok sisipan cepat untuk identitas warga, data usaha, judul surat, tanda tangan, penutup, dan tembusan.
- Validasi placeholder tidak dikenal agar salah ketik seperti `{{nama_wargaa}}` mudah ditemukan.
- Peringatan placeholder penting yang belum ada, terutama `{{nomor_surat}}` dan `{{tanggal_surat}}`.
- Preview A4 langsung saat mengedit template.
- Tombol bersihkan format untuk konten hasil paste dari Word.
- Reset template ke versi bawaan.
- Toolbar tabel dasar: sisipkan tabel, tambah/hapus baris, tambah/hapus kolom, hapus tabel.
- Duplikasi template dari template yang sudah ada.
- Pencarian placeholder dalam editor.
- Mode layar penuh untuk template panjang.
- Toolbar tabel lanjutan: merge cell dan split cell.
- Sisipan kop surat ke dalam isi template bila admin ingin kop menjadi bagian template, bukan hanya preview.
- Deteksi konten yang kemungkinan terpotong saat print.
- Deteksi tabel terlalu lebar untuk kertas A4.
- Riwayat versi lokal untuk beberapa perubahan terakhir.
- Autosave draft sebelum klik simpan.
- Zoom preview A4.
- Catatan internal template yang tidak ikut tercetak.
- Export/import template sebagai file JSON.
- Shortcut keyboard untuk admin yang sudah terbiasa.

## Sisa Wishlist

- Semua item wishlist awal sudah diimplementasikan.
